This class project was an entirely self-directed passion project, with the only requirement being that it addressed some kind of problem. I addressed the topic of genre classification, which to me is confused and often fails to accurately represent music. My project was a data visualization system for music, through which you could transform data values for tempo, volume, duration, key signature, and time signature of any song into a visual form. This document thoroughly details the process used to general these visual forms.
