This class project mapping a given theme, being electrical generation and distribution, onto the framework of an existing board game. My partner for the project and I chose to base ours off of the game Caesar!: Seize Rome in 20 Minutes! We maintained most of the core ideas of the game, but reimagined the game board and added the theme of electrical power to all of the elements.
